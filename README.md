## Features

- Add notes (title + body)
- Delete notes
- **Pin/unpin** notes (pinned notes stay at the top)
- Search notes
- Responsive layout (sidebar collapses on mobile)
- Local persistence via `localStorage`

## Run locally

```bash
npm install
npm run dev
```

Then open the dev server URL shown in your terminal.
